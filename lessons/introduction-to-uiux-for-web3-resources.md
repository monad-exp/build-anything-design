# Resources — Introduction to UI/UX Design for Web3

- [Uniswap](https://app.uniswap.org) — study swap flows, approvals, and transaction feedback
- [Phantom](https://phantom.app) — study wallet connection, onboarding, and signing UX
- [Coinbase](https://www.coinbase.com) — study trust signals and simplified language for mainstream users
- [Jupiter](https://jup.ag) — study how complex trading UI can stay approachable
- [Nielsen Norman Group — UX vs UI](https://www.nngroup.com/articles/ux-vs-ui/) — quick reference on the core distinction covered in this lesson
- [Nielsen Norman Group — Blockchain User Experience](https://www.nngroup.com/articles/blockchain-user-experience/) — Web3-specific UX patterns for wallets, approvals, and transaction states
- [WCAG](https://www.w3.org/WAI/standards-guidelines/wcag/) — accessibility standards for contrast, labels, keyboard navigation, and screen readers
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) — check color contrast before shipping UI
